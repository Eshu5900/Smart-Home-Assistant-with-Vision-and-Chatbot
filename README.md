# 🏠 Smart Home Assistant with Vision and Chatbot

## 📌 Description
This project implements a Smart Home Assistant by integrating **computer vision** and a **rule-based chatbot**.  
The system detects objects in an image using YOLO (OpenCV) and responds to user queries based on the detected objects.

---

## 🚀 Features
- Object detection using YOLO (COCO dataset)
- Detection of multiple objects in a scene
- Chatbot-based interaction
- Context-aware responses
- Continuous user interaction

---

## 🛠️ Technologies Used
- Python  
- OpenCV  
- YOLOv3  
- NumPy  

---

## ⚙️ How It Works
1. Upload an image of a home environment  
2. Perform object detection using YOLO  
3. Extract detected object labels  
4. Ask questions using chatbot  
5. Get responses based on detected objects  

---

## ▶️ Example Queries
- What do you see?  
- Is there a chair?  
- How many objects are present?  
- Is anyone in the room?
